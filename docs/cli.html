CLI Commands

Artisan command for reading, writing, and previewing XFA PDFs from the terminal.

xfa-pdf:manage

php artisan xfa-pdf:manage {filename} [options]

Arguments

ArgumentDescription
filenamePath to the XFA PDF file (required)

Options

OptionDescription
--readRead and display all XFA datasets
--rawOutput raw XML (combine with --read)
--sectionsList all section names
--section=NAMEDisplay fields for a specific section
--get=PATHGet a field value by slash-separated path
--set=PATH=VALUESet a field value
--output=FILEOutput file path (default: overwrites original)
--previewGenerate HTML preview and open in browser

Examples

List sections

php artisan xfa-pdf:manage form.pdf --sections

Read all data

# Formatted output
php artisan xfa-pdf:manage form.pdf --read

# Raw XML
php artisan xfa-pdf:manage form.pdf --read --raw

View a specific section

php artisan xfa-pdf:manage form.pdf --section=Section_3_AppraisalDetails

Get a single field

php artisan xfa-pdf:manage form.pdf \
    --get="Section_3_AppraisalDetails/appraisalYear"

Set a field value

# Set and overwrite original
php artisan xfa-pdf:manage form.pdf \
    --set="Section_3_AppraisalDetails/appraisalYear=2025"

# Set and save to a new file
php artisan xfa-pdf:manage form.pdf \
    --set="Section_3_AppraisalDetails/appraisalYear=2025" \
    --output=output.pdf

Generate preview

php artisan xfa-pdf:manage form.pdf --preview
