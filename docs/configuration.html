Configuration

After publishing, the config file is at config/xfa-pdf.php:

<?php

return [
    // URL prefix for all XFA PDF routes
    'route_prefix' => 'xfa-pdf',

    // Middleware applied to all routes
    'middleware' => ['web'],

    // Filesystem disk for storing uploaded PDFs
    'disk' => 'local',

    // Directory within the disk for uploads
    'upload_path' => 'xfa-pdf',

    // Maximum upload file size in MB
    'max_upload_size' => 20,
];

Options Reference

KeyDefaultDescription
route_prefix 'xfa-pdf' URL prefix for all package routes (e.g. /xfa-pdf, /xfa-pdf/upload)
middleware ['web'] Middleware stack applied to all routes. Add 'auth' to require authentication.
disk 'local' Laravel filesystem disk for PDF file storage
upload_path 'xfa-pdf' Subdirectory within the disk where uploaded PDFs are stored
max_upload_size 20 Maximum upload file size in megabytes

Examples

Restrict routes to authenticated users

'middleware' => ['web', 'auth'],

Change the URL prefix

'route_prefix' => 'admin/forms',
// Routes: /admin/forms, /admin/forms/upload, etc.

Use S3 storage

'disk' => 's3',
'upload_path' => 'xfa-pdfs',
