Exceptions

All exceptions extend XfaPdfException and use static factory methods with descriptive messages.

InvalidPdfException

Thrown when the PDF file is invalid, missing, or unreadable.

static fileNotFound(string $path): self

static unreadable(string $path): self

static invalidFormat(): self

NoXfaDataException

Thrown when the PDF does not contain valid XFA data.

static noDatasetsFound(): self

static noTemplateFound(): self

static failedToParse(): self

FieldNotFoundException

Thrown when a field path or repeatable container cannot be found.

static atPath(string $fieldPath): self

static containerNotFound(string $sectionName, string $containerName): self

Usage Example

use Xfa\Pdf\Facades\XfaPdf;
use Xfa\Pdf\Exceptions\InvalidPdfException;
use Xfa\Pdf\Exceptions\NoXfaDataException;
use Xfa\Pdf\Exceptions\FieldNotFoundException;

try {
    XfaPdf::setFile('/path/to/file.pdf');
    $value = XfaPdf::readField('Section_3', 'missingField');
} catch (InvalidPdfException $e) {
    // File not found, unreadable, or invalid format
    logger()->error($e->getMessage());
} catch (NoXfaDataException $e) {
    // PDF has no XFA datasets or template
    logger()->error($e->getMessage());
} catch (FieldNotFoundException $e) {
    // Field path doesn't exist in datasets
    logger()->error($e->getMessage());
}
