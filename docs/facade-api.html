Facade API

The XfaPdf facade provides a stateful API. Call setFile() first, then chain operations.

use Xfa\Pdf\Facades\XfaPdf;

Loading

setFile(string $filePath): self

Load an XFA PDF file. Must be called before any other method.

Reading Data

sections(): array

Get all section names. Returns string[].

read(string $sectionName): array

Read all fields in a section.

readField(string $sectionName, string $fieldName): ?string

Read a single field value.

rawXml(): string

Get the raw XFA datasets XML string.

toArray(): array

Convert entire datasets to a nested PHP array.

fieldMetadata(): array

Get field metadata (type, options, caption) from the template.

repeatableSubforms(): array

Get repeatable subform definitions.

navigationSections(): array

Get section labels from template MenuBar tooltips. Returns array<int, string>.

Writing Data

update(string $sectionName, array $fields): self

Update multiple fields. $fields is ['fieldName' => 'value', ...].

updateField(string $sectionName, string $fieldName, string $value): self

Update a single field.

save(?string $outputPath = null): bool

Save changes via incremental update. null overwrites the original file.

Preview

preview(): string

Generate a complete HTML preview with all field types as HTML form controls.

Service Accessors

Access underlying services for advanced use:

$manager = app('xfa-pdf');

$manager->pdf();         // PdfBinaryService
$manager->datasets();    // DatasetService
$manager->templates();   // TemplateService
$manager->repeatables(); // RepeatableService
$manager->previews();    // PreviewService
$manager->namespaces();  // NamespaceService

Section Proxy

The Section class provides a fluent interface scoped to a single section.

read(): array

Read all fields in this section.

readField(string $fieldName): ?string

Read a single field.

update(array $fields): self

Update multiple fields. Chainable.

updateField(string $fieldName, string $value): self

Update a single field. Chainable.

getName(): string

Get the XFA section name (e.g. Section_3_AppraisalDetails).

$section = XfaPdf::personalDetails();

$data = $section->read();
echo $data['appraisalYear']; // '2024'

$section->updateField('appraisalYear', '2025')
        ->updateField('contactEmail', 'new@example.com');

XfaPdf::save();
