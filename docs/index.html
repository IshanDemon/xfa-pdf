XFA-PDF Package

A Laravel package for reading, editing, previewing, and saving XFA (XML Forms Architecture) PDFs — entirely in pure PHP.

Pure PHP

No paid dependencies. Uses DOMDocument for XML and raw binary parsing for PDF.

Non-Destructive

Incremental PDF updates append changes without modifying original content.

Full Stack

PHP API, Facade, Artisan CLI, and Web UI with upload/preview/edit/download.

Key Features

  • Load and parse XFA PDFs — extract datasets and template XML
  • Read and write individual fields or entire sections
  • Add, remove, and update repeatable subform items
  • Generate interactive HTML previews with correct field types
  • Save changes back to PDF via incremental binary updates
  • Auto-detect XML namespaces (works with any XFA form)
  • Section-based navigation mirroring original XFA form structure
  • Web UI with upload, preview, edit, and download
  • Artisan CLI for scripted/batch operations
  • Fluent Section API for clean, readable code

Quick Example

use Xfa\Pdf\Facades\XfaPdf;

// Load, read, update, save
XfaPdf::setFile('form.pdf');

$year = XfaPdf::readField('Section_3_AppraisalDetails', 'appraisalYear');

XfaPdf::update('Section_3_AppraisalDetails', [
    'appraisalYear' => '2025',
    'appriseeName'  => 'Dr Jane Smith',
]);

XfaPdf::save('/path/to/output.pdf');
Note: XFA is deprecated in PDF 2.0. Only Adobe Acrobat Reader fully renders XFA forms. Browser PDF viewers do not support XFA.
