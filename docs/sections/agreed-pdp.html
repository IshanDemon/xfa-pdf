Section 18: The Agreed PDP

This section captures the agreed Personal Development Plan goals set during the appraisal meeting. Each goal is a repeatable item with a role and details.

Section name: Section_18_TheAgreedPDP
Fluent accessor: XfaPdf::agreedPdp()
Navigation label: The agreed personal development plan

What's in this section?

FieldTypeWhat it is
pdpGoals > goalRepeatablePDP goal rows, each with a role and details

Each goal row inside pdpGoals contains: role (select — the relevant area of practice), details (textarea — detail of the PDP goal including what will be done, how, and by when).

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Reading existing data

Read all fields in this section at once:

// Read everything in The Agreed PDP
$data = XfaPdf::read('Section_18_TheAgreedPDP');

// Or using the fluent accessor
$data = XfaPdf::agreedPdp()->read();

// $data looks like:
// [
//     'pdpGoals' => [
//         'goal' => [
//             [
//                 'role' => 'Clinical',
//                 'details' => 'Complete Advanced Life Support recertification by March 2025...',
//             ],
//             [
//                 'role' => 'Educational',
//                 'details' => 'Develop skills in musculoskeletal medicine through supervised clinics...',
//             ],
//         ],
//     ],
// ]

Read a single field:

$goals = XfaPdf::agreedPdp()->readField('pdpGoals');
// Returns the nested array of goal rows

PDP Goals (Repeatable Table)

Each row in the PDP goals table represents one development goal agreed during the appraisal meeting. Rows are stored as goal elements inside the pdpGoals container. Each row has two fields:

Row fieldTypeDescription
roleSelectThe area of practice or role the goal relates to (e.g. Clinical, Educational, Leadership)
detailsTextareaFull description of the PDP goal, including what will be done, how, and target completion date

Adding a new PDP goal

To add a new goal row:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Add a PDP goal
$xfaPdf = $manager->addRepeatableItem(
    $xfaPdf,
    'Section_18_TheAgreedPDP',        // section name
    'pdpGoals',                        // container name
    [
        'role'    => 'Clinical',
        'details' => 'Complete Advanced Life Support recertification by March 2025. '
                   . 'Attend the two-day ALS provider course at Manchester Royal '
                   . 'Infirmary. This will maintain emergency response competence '
                   . 'and is a requirement for my out-of-hours clinical role.',
    ]
);

$manager->saveXfaPdf($xfaPdf);

Replacing all PDP goals

If you want to clear existing goals and set new ones from scratch, use setRepeatableItems():

$xfaPdf = $manager->setRepeatableItems(
    $xfaPdf,
    'Section_18_TheAgreedPDP',        // section
    'pdpGoals',                        // container
    'goal',                            // element name
    [
        [
            'role'    => 'Clinical',
            'details' => 'Complete Advanced Life Support recertification by March 2025. '
                       . 'Attend the two-day ALS provider course at Manchester Royal '
                       . 'Infirmary. This will maintain emergency response competence '
                       . 'and is a requirement for my out-of-hours clinical role.',
        ],
        [
            'role'    => 'Clinical',
            'details' => 'Develop skills in musculoskeletal medicine through supervised '
                       . 'clinics. Attend 10 supervised MSK clinics at Trafford General '
                       . 'Hospital with Dr Patel (consultant rheumatologist) between '
                       . 'April and September 2025. Complete the online PGCert module '
                       . 'in MSK Ultrasound by December 2025.',
        ],
        [
            'role'    => 'Educational',
            'details' => 'Obtain GP Trainer accreditation by completing the Trainer '
                       . 'Approval process. Submit portfolio by June 2025, host training '
                       . 'practice visit in autumn 2025. This supports the practice\'s '
                       . 'commitment to medical education and workforce development.',
        ],
        [
            'role'    => 'Leadership',
            'details' => 'Lead the practice-wide implementation of structured medication '
                       . 'reviews for patients on 10 or more repeat medications. Develop '
                       . 'a protocol by May 2025, train all prescribers by July 2025, '
                       . 'and audit outcomes at 6 months. Aligns with PCN prescribing '
                       . 'safety priorities.',
        ],
    ]
);

$manager->saveXfaPdf($xfaPdf);

Reading existing PDP goals

$goals = $manager->getRepeatableItems(
    $xfaPdf,
    'Section_18_TheAgreedPDP',
    'pdpGoals'
);

foreach ($goals as $index => $goal) {
    echo "Goal {$index}: [{$goal['role']}] {$goal['details']}\n\n";
}
// Goal 0: [Clinical] Complete Advanced Life Support recertification by March 2025...
// Goal 1: [Clinical] Develop skills in musculoskeletal medicine through supervised clinics...
// Goal 2: [Educational] Obtain GP Trainer accreditation by completing the Trainer Approval process...
// Goal 3: [Leadership] Lead the practice-wide implementation of structured medication reviews...

Removing a PDP goal

Remove a goal by its index (0-based):

// Remove the third goal (index 2)
$xfaPdf = $manager->removeRepeatableItem(
    $xfaPdf,
    'Section_18_TheAgreedPDP',
    'pdpGoals',
    2
);

$manager->saveXfaPdf($xfaPdf);

Complete Controller Example

Here's a full example filling the entire Agreed PDP section in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    public function fillAgreedPdp()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        $manager = app('xfa-pdf');
        $xfaPdf = $manager->load($pdfPath);

        // -----------------------------------------------
        // 1. Set PDP goal rows (this section is entirely
        //    made up of repeatable goal rows)
        // -----------------------------------------------
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_18_TheAgreedPDP',
            'pdpGoals',
            'goal',
            [
                [
                    'role'    => 'Clinical',
                    'details' => 'Complete Advanced Life Support '
                               . 'recertification by March 2025. Attend '
                               . 'the two-day ALS provider course at '
                               . 'Manchester Royal Infirmary. This will '
                               . 'maintain emergency response competence '
                               . 'and is a requirement for the out-of-hours '
                               . 'clinical role.',
                ],
                [
                    'role'    => 'Clinical',
                    'details' => 'Develop skills in musculoskeletal '
                               . 'medicine through supervised clinics. '
                               . 'Attend 10 supervised MSK clinics at '
                               . 'Trafford General Hospital with Dr Patel '
                               . '(consultant rheumatologist) between April '
                               . 'and September 2025. Complete the online '
                               . 'PGCert module in MSK Ultrasound by '
                               . 'December 2025.',
                ],
                [
                    'role'    => 'Educational',
                    'details' => 'Obtain GP Trainer accreditation by '
                               . 'completing the Trainer Approval process. '
                               . 'Submit portfolio by June 2025, host '
                               . 'training practice visit in autumn 2025. '
                               . 'This supports the practice\'s commitment '
                               . 'to medical education and workforce '
                               . 'development.',
                ],
                [
                    'role'    => 'Leadership',
                    'details' => 'Lead the practice-wide implementation '
                               . 'of structured medication reviews for '
                               . 'patients on 10 or more repeat medications. '
                               . 'Develop a protocol by May 2025, train all '
                               . 'prescribers by July 2025, and audit '
                               . 'outcomes at 6 months. Aligns with PCN '
                               . 'prescribing safety priorities.',
                ],
            ]
        );

        // -----------------------------------------------
        // 2. Save the PDF
        // -----------------------------------------------
        $manager->saveXfaPdf($xfaPdf);
        // Or: XfaPdf::save();

        return redirect()->back()->with('success', 'Agreed PDP saved.');
    }
}
