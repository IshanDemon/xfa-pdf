Section 17: Appraisal Checklist

A checklist of 20 confirmation items that the doctor and appraiser must tick before completing the appraisal. All fields are checkboxes using "1" for checked and "0" for unchecked.

Section name: Section_17_Checklist
Fluent accessor: XfaPdf::checklist()
Navigation label: Appraisal checklist

What's in this section?

The checklist is divided into two groups of items, followed by a final agreement checkbox. All 20 fields are checkboxes.

This section has no repeatable items and no conditional fields. All 20 fields are standalone checkboxes.

Checkbox values: XFA checkboxes in this form use "1" for checked and "0" for unchecked. These are string values — always pass "1" or "0" as strings, not boolean true/false or integer 1/0.

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Reading existing data

Read all fields in this section at once:

// Read everything in the Checklist
$data = XfaPdf::read('Section_17_Checklist');

// Or using the fluent accessor
$data = XfaPdf::checklist()->read();

// $data looks like:
// [
//     'section1_a' => '1',
//     'section1_b' => '1',
//     'section1_c' => '0',
//     'section1_d' => '1',
//     'section2_a' => '1',
//     'section2_b' => '0',
//     'section2_c' => '0',
//     ...
//     'section2_o' => '0',
//     'agreementChk' => '0',
// ]

Read a single checkbox:

$isChecked = XfaPdf::checklist()->readField('section1_a');
// '1' (checked) or '0' (unchecked)

$agreed = XfaPdf::checklist()->readField('agreementChk');
// '1' or '0'

Updating Checkboxes

Each checkbox accepts "1" (checked) or "0" (unchecked). You can update them one at a time or in bulk.

Setting a single checkbox

To check or uncheck an individual item:

// Check the first pre-appraisal item
XfaPdf::checklist()->updateField('section1_a', '1');

// Uncheck the final agreement
XfaPdf::checklist()->updateField('agreementChk', '0');

XfaPdf::save();

Bulk update — checking all items

To check all 20 checklist items at once, pass them all in a single update() call:

// Check all pre-appraisal items
XfaPdf::checklist()->update([
    'section1_a' => '1',
    'section1_b' => '1',
    'section1_c' => '1',
    'section1_d' => '1',
]);

XfaPdf::save();

Or check all appraisal meeting items:

// Check all appraisal meeting items

Tip: You can build the checklist array dynamically from a form submission. For example, if your HTML form uses checkbox inputs with the same field names, you can map the submitted values to "1" or "0" and pass the entire array to update().

Complete Controller Example

Here's a full example filling the Checklist section from a form submission in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    /**
     * All checkbox field names in this section.
     */
    private const CHECKLIST_FIELDS = [
        'section1_a', 'section1_b', 'section1_c', 'section1_d',
        'section2_a', 'section2_b', 'section2_c', 'section2_d',
        'section2_e', 'section2_f', 'section2_g', 'section2_h',
        'section2_i', 'section2_j', 'section2_k', 'section2_l',
        'section2_m', 'section2_n', 'section2_o',
        'agreementChk',
    ];

    public function fillChecklist(Request $request)
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        XfaPdf::setFile($pdfPath);

        // -----------------------------------------------
        // 1. Build the checkbox values from form input.
        //    HTML checkboxes only submit when checked,
        //    so we default to "0" for unchecked items.
        // -----------------------------------------------
        $checklistData = [];
        foreach (self::CHECKLIST_FIELDS as $field) {
            $checklistData[$field] = $request->has($field) ? '1' : '0';
        }

        // -----------------------------------------------
        // 2. Update all checkboxes in one call
        // -----------------------------------------------
        XfaPdf::checklist()->update($checklistData);

        // -----------------------------------------------
        // 3. Save the PDF
        // -----------------------------------------------
        XfaPdf::save();

        return redirect()->back()->with(
            'success',
            'Checklist saved.'
        );
    }

    /**
     * Read and display existing Checklist data.
     */
    public function showChecklist()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        XfaPdf::setFile($pdfPath);

        $data = XfaPdf::checklist()->read();

        // Convert "1"/"0" strings to booleans for Blade
        $checkboxes = [];
        foreach (self::CHECKLIST_FIELDS as $field) {
            $checkboxes[$field] = ($data[$field] ?? '0') === '1';
        }

        return view('appraisal.checklist', [
            'checkboxes' => $checkboxes,
        ]);
    }
}
