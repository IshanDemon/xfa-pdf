Section 7: Continuing Professional Development (CPD)

This section records the doctor's CPD activities, including whether they belong to a Royal College faculty, whether they hold a CPD certificate, and a detailed table of CPD activities with descriptions, outcomes, and credits.

Section name: Section_7_CPD
Fluent accessor: XfaPdf::cpd()

What's in this section?

FieldTypeWhat it is
memberOfFacultyRadio (yes/no)Member of a Royal College faculty? (conditional trigger)
annualCertificateRadio (yes/no)Has annual CPD certificate? (shown when memberOfFaculty = "yes")
annualCertAttachmentNestedAttachment for certificate (attachmentID, fileName)
additionalCPDActivitiesRadio (yes/no)Has additional CPD record? (shown when memberOfFaculty = "yes")
additionalCPDAttachmentNestedAttachment for additional CPD record (attachmentID, fileName)
CPDActivitiesRadio (yes/no)Has own CPD record? (shown when memberOfFaculty = "no")
CPDAttachmentNestedAttachment for own CPD record (attachmentID, fileName)
commentaryOnActivitiesTextareaCommentary on how CPD supports scope of work
cpdList > cpdRepeatableCPD activity rows (role, description, outcome, credits, supportLocation)
appraiserCommentsTextareaAppraiser comments (mirrors to Section 19)
Conditional fields: The memberOfFaculty radio acts as a trigger. When set to "yes", the fields annualCertificate and additionalCPDActivities become visible. When set to "no", the field CPDActivities becomes visible instead. Your code should set the appropriate branch of fields based on the radio value.

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Reading existing data

Read all fields in this section at once:

// Read everything in CPD
$data = XfaPdf::read('Section_7_CPD');

$data = XfaPdf::cpd()->read();

CPD Activities (Repeatable Table)

Each row in the CPD table represents one CPD activity. The repeatable element cpd lives inside the cpdList container. Each row has these fields:

Row FieldTypeDescription
roleSelectThe role this CPD relates to (e.g. "GP", "GP Trainer")
descriptionTextareaDescription of the CPD activity
outcomeTextareaWhat was learned or changed as a result
creditsNumberCPD credits awarded
supportLocationSelect"Attached", "Emailed to appraiser", "Provided separately", or "Not available"

Adding a new CPD activity

To add a new row with all fields filled in:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Add a CPD activity row
$xfaPdf = $manager->addRepeatableItem(
    $xfaPdf,
    'Section_7_CPD',        // section name
    'cpdList',              // container name
    [
        'role'            => 'GP',
        'description'     => 'RCGP Annual Conference 2024 - Attended three-day conference covering advances in primary care, diabetes management, and mental health.',
        'outcome'         => 'Updated knowledge on NICE guidelines for Type 2 diabetes. Implemented new HbA1c monitoring protocol at practice.',
        'credits'         => '12',
        'supportLocation' => 'Attached',
    ]
);

$manager->saveXfaPdf($xfaPdf);

Reading existing CPD activities

$rows = $manager->getRepeatableItems(
    $xfaPdf,
    'Section_7_CPD',
    'cpdList'
);

foreach ($rows as $index => $row) {
    echo "Activity {$index}: {$row['description']} ({$row['credits']} credits)\n";
}
// Activity 0: RCGP Annual Conference 2024 - Three-day conference... (12 credits)
// Activity 1: Safeguarding Level 3 Training - Full-day mandatory... (6 credits)
// Activity 2: BMJ Learning Module: Chronic Pain Management... (4 credits)

// Calculate total credits
$totalCredits = array_sum(array_column($rows, 'credits'));
echo "Total CPD credits: {$totalCredits}\n";
// Total CPD credits: 31

Removing a CPD activity

Remove a row by its index (0-based):

// Remove the third activity (index 2)
$xfaPdf = $manager->removeRepeatableItem(
    $xfaPdf,
    'Section_7_CPD',
    'cpdList',
    2
);

$manager->saveXfaPdf($xfaPdf);

Radio / Conditional Fields

The memberOfFaculty radio controls which fields are visible in the XFA form. When you fill data programmatically, you should set the appropriate branch of fields based on the radio value.

When the doctor IS a faculty member

Trigger: memberOfFaculty = "yes"

When the doctor is a member of a Royal College faculty, the form shows questions about their annual CPD certificate and any additional CPD activities.

XfaPdf::cpd()->update([
    // Set the trigger radio
    'memberOfFaculty' => 'yes',

    // These fields appear when memberOfFaculty = "yes"
    'annualCertificate' => 'yes',
    'additionalCPDActivities' => 'yes',
]);

// If you need to set attachment metadata for the certificate
XfaPdf::cpd()->update([
    'annualCertAttachment/attachmentID' => 'cert-2024-001',
    'annualCertAttachment/fileName' => 'RCGP_CPD_Certificate_2024.pdf',
]);

// And for the additional CPD record
XfaPdf::cpd()->update([
    'additionalCPDAttachment/attachmentID' => 'cpd-extra-001',
    'additionalCPDAttachment/fileName' => 'Additional_CPD_Record_2024.pdf',
]);

When the doctor is NOT a faculty member

Trigger: memberOfFaculty = "no"

When the doctor is not a faculty member, the form shows a different question about whether they maintain their own CPD record.

XfaPdf::cpd()->update([
    // Set the trigger radio
    'memberOfFaculty' => 'no',

    // This field appears when memberOfFaculty = "no"
    'CPDActivities' => 'yes',
]);

// Attach the CPD record
XfaPdf::cpd()->update([
    'CPDAttachment/attachmentID' => 'cpd-own-001',
    'CPDAttachment/fileName' => 'Personal_CPD_Log_2024.pdf',
]);
Important: When reading data, both branches of conditional fields will be present in the returned array. Check the value of memberOfFaculty to determine which branch is relevant. The inactive branch will typically contain empty values.

Text Fields

Below the CPD table and radio questions, there are two free-text fields. These are standalone (not repeatable).

Commentary on CPD activities

Field: commentaryOnActivities

This is where the doctor explains how their CPD activities relate to their scope of work. The form label reads: "Please provide a brief commentary on how these CPD activities have supported you in your various roles."

XfaPdf::cpd()->updateField(
    'commentaryOnActivities',
    'My CPD activities this year have been closely aligned with my scope '
    . 'of work. The RCGP conference and BMJ learning modules directly '
    . 'support my clinical GP role, particularly in diabetes and chronic '
    . 'pain management where I have seen measurable improvements in patient '
    . 'outcomes. The safeguarding training is essential for my GP Trainer '
    . 'role, ensuring I can properly supervise registrars. The respiratory '
    . 'webinar series has been invaluable for my PCN Clinical Lead work, '
    . 'leading to a 15% reduction in emergency respiratory admissions across '
    . 'the network. The significant event audit has strengthened our whole '
    . 'practice approach to sepsis identification.'
);

Appraiser's comments

Field: appraiserComments

Written by the appraiser, summarising their view of the doctor's CPD activities. This field mirrors to Section 19 (Appraisal Summary).

XfaPdf::cpd()->updateField(
    'appraiserComments',
    'Dr Smith has undertaken a comprehensive and well-targeted CPD programme '
    . 'this year totalling 31 credits. Activities are clearly mapped to their '
    . 'scope of work across all three roles. The impact of the respiratory '
    . 'webinar series on PCN emergency admissions is particularly noteworthy. '
    . 'I would encourage continued focus on chronic disease management CPD '
    . 'and further development of the sepsis screening protocol.'
);

Complete Controller Example

Here's a full example filling the entire CPD section in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    public function fillCpd()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        $manager = app('xfa-pdf');
        $xfaPdf = $manager->load($pdfPath);

        // -----------------------------------------------
        // 1. Set radio / conditional fields
        // -----------------------------------------------
        XfaPdf::setFile($pdfPath);

        XfaPdf::cpd()->update([
            'memberOfFaculty' => 'yes',
            'annualCertificate' => 'yes',
            'additionalCPDActivities' => 'yes',
        ]);

        // Attachment metadata
        XfaPdf::cpd()->update([
            'annualCertAttachment/attachmentID' => 'cert-2024-001',
            'annualCertAttachment/fileName' => 'RCGP_CPD_Certificate_2024.pdf',
            'additionalCPDAttachment/attachmentID' => 'cpd-extra-001',
            'additionalCPDAttachment/fileName' => 'Additional_CPD_Record_2024.pdf',
        ]);

        // -----------------------------------------------
        // 2. Set CPD activity rows (repeatable)
        // -----------------------------------------------
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_7_CPD',
            'cpdList',
            'cpd',
            [
                [
                    'role'            => 'GP',
                    'description'     => 'RCGP Annual Conference 2024 - Three-day '
                                       . 'conference covering primary care advances, '
                                       . 'diabetes management, and mental health.',
                    'outcome'         => 'Updated diabetes management protocols. '
                                       . 'Implemented new HbA1c monitoring approach '
                                       . 'at practice level.',
                    'credits'         => '12',
                    'supportLocation' => 'Attached',
                ],
                [
                    'role'            => 'GP Trainer',
                    'description'     => 'Safeguarding Level 3 Training - Full-day '
                                       . 'mandatory safeguarding course at the Deanery.',
                    'outcome'         => 'Refreshed child protection referral pathways '
                                       . 'and documentation requirements. Updated '
                                       . 'practice safeguarding policy.',
                    'credits'         => '6',
                    'supportLocation' => 'Emailed to appraiser',
                ],
                [
                    'role'            => 'GP',
                    'description'     => 'BMJ Learning Module: Chronic Pain Management '
                                       . 'in Primary Care.',
                    'outcome'         => 'Improved understanding of non-pharmacological '
                                       . 'approaches. Started referring patients to '
                                       . 'local pain management programme.',
                    'credits'         => '4',
                    'supportLocation' => 'Attached',
                ],
                [
                    'role'            => 'Clinical Lead - PCN',
                    'description'     => 'NHS England Respiratory Webinar Series '
                                       . '(6 sessions over 3 months).',
                    'outcome'         => 'Applied updated asthma pathway across the '
                                       . 'PCN. Reduced emergency respiratory admissions '
                                       . 'by 15% in Q3.',
                    'credits'         => '6',
                    'supportLocation' => 'Provided separately',
                ],
                [
                    'role'            => 'GP',
                    'description'     => 'Practice-based significant event audit on '
                                       . 'missed sepsis diagnosis.',
                    'outcome'         => 'Developed practice-wide sepsis screening '
                                       . 'checklist now used in all urgent appointments.',
                    'credits'         => '3',
                    'supportLocation' => 'Attached',
                ],
            ]
        );

        // -----------------------------------------------
        // 3. Fill text fields
        // -----------------------------------------------
        XfaPdf::cpd()->update([
            'commentaryOnActivities' =>
                'My CPD activities this year have been closely aligned with '
                . 'my scope of work. The RCGP conference and BMJ modules '
                . 'directly support my clinical GP role, particularly in '
                . 'diabetes and chronic pain management. The safeguarding '
                . 'training is essential for my GP Trainer role. The '
                . 'respiratory webinar series has been invaluable for my '
                . 'PCN Clinical Lead work, leading to a 15% reduction in '
                . 'emergency respiratory admissions. The significant event '
                . 'audit strengthened our practice approach to sepsis.',

            'appraiserComments' =>
                'Dr Smith has undertaken a comprehensive CPD programme '
                . 'totalling 31 credits, well-targeted across all three '
                . 'roles. The impact on PCN respiratory admissions is '
                . 'particularly noteworthy. I would encourage continued '
                . 'focus on chronic disease management and further '
                . 'development of the sepsis screening protocol.',
        ]);

        // -----------------------------------------------
        // 4. Save the PDF
        // -----------------------------------------------
        $manager->saveXfaPdf($xfaPdf);
        // Or: XfaPdf::save();

        return redirect()->back()->with('success', 'CPD section saved.');
    }
}
