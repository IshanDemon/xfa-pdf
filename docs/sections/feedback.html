Section 10: Colleague and Patient Feedback

This section records colleague and patient feedback. It has two halves: colleague feedback and patient feedback, each with its own conditional radio, date, comments, and repeatable table of feedback items.

Section name: Section_10_Feedback
Fluent accessor: XfaPdf::feedback()

What's in this section?

Colleague Feedback

FieldTypeWhat it is
colleagueFeedbackAvailableRadio (yes/no)Whether colleague feedback is available (conditional trigger)
lastColleagueFeedbackDateMultiline textDate of last colleague feedback (shown when "no")
colleagueFeedbackCommentMultiline textComment about colleague feedback (shown when "no")
colleague > feedbackRowRepeatableColleague feedback rows (shown when "yes" or "no")
whyNoColleagueFeedbackTextWhy no colleague feedback is available

Patient Feedback

FieldTypeWhat it is
patientFeedbackAvailableRadio (yes/no)Whether patient feedback is available (conditional trigger)
lastPatientFeedbackDateMultiline textDate of last patient feedback (shown when "no")
patientFeedbackCommentMultiline textComment about patient feedback (shown when "no")
patient > feedbackRowRepeatablePatient feedback rows (shown when "yes" or "no")
whyNoPatientTextWhy no patient feedback is available

Shared Fields

FieldTypeWhat it is
actionsTakenMultiline textCommentary on feedback and actions taken (shown when "yes")
appraiserCommentsMultiline textAppraiser comments (mirrors to Section 19)

Each feedbackRow contains: role (select), activity (textarea — date and description), outcome (textarea), supportLocation (select), firstEntry (text), amendData (text).

Conditional fields: This section has two independent triggers that control which fields are visible.

1. colleagueFeedbackAvailable
• "yes": shows colleague feedbackRow table and actionsTaken
• "no": shows colleague feedbackRow table, lastColleagueFeedbackDate, and colleagueFeedbackComment

2. patientFeedbackAvailable
• "yes": shows patient feedbackRow table and actionsTaken
• "no": shows patient feedbackRow table, lastPatientFeedbackDate, and patientFeedbackComment

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Colleague Feedback

The colleague feedback half of this section starts with a yes/no radio that controls which fields are visible. When feedback is available, the repeatable table and actions commentary appear. When it is not, the date and comment fields appear instead.

Setting the colleague feedback radio

Field: colleagueFeedbackAvailable (radio — conditional trigger)

Set whether colleague feedback is available. This controls which fields are visible in the XFA form.

// Colleague feedback IS available
XfaPdf::feedback()->updateField('colleagueFeedbackAvailable', 'yes');

// Colleague feedback is NOT available
XfaPdf::feedback()->updateField('colleagueFeedbackAvailable', 'no');

Conditional fields (when "no")

When colleague feedback is not available, fill in the date and comment fields:

XfaPdf::feedback()->updateField('colleagueFeedbackAvailable', 'no');

XfaPdf::feedback()->updateField(
    'lastColleagueFeedbackDate',
    'Last colleague feedback was collected in March 2022 as part of '
    . 'the previous appraisal cycle.'
);

XfaPdf::feedback()->updateField(
    'colleagueFeedbackComment',
    'Colleague feedback was due in March 2024 but was delayed due to '
    . 'practice restructuring. New questionnaires have been distributed '
    . 'and results are expected by June 2024.'
);

Colleague feedback rows (repeatable)

Each row represents one colleague who provided feedback. The table appears regardless of the radio selection. Fields per row: role (select), activity (textarea), outcome (textarea), supportLocation (select), firstEntry (text), amendData (text).

Adding a colleague feedback row

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Add a colleague feedback row
$xfaPdf = $manager->addRepeatableItem(
    $xfaPdf,
    'Section_10_Feedback',     // section name
    'colleague',               // container name
    [
        'role'            => 'Consultant',
        'activity'        => 'March 2024 - GMC colleague questionnaire distributed '
                           . 'to 20 colleagues across primary and secondary care. '
                           . '18 responses received (90% response rate).',
        'outcome'         => 'Overall very positive feedback. Mean score 4.6/5 across '
                           . 'all domains. Colleagues highlighted strong communication '

                           . 'timeliness of referral letters.',
        'supportLocation' => 'Yes',
        'firstEntry'      => '',
        'amendData'       => '',
    ]
);

$manager->saveXfaPdf($xfaPdf);

    $xfaPdf,

Removing a colleague feedback row

Remove a row by its index (0-based):

// Remove the second colleague feedback row (index 1)

    $xfaPdf,
    'Section_10_Feedback',
    'colleague',
    1
);

$manager->saveXfaPdf($xfaPdf);

Patient Feedback

The patient feedback half mirrors the colleague feedback structure. It has its own independent radio trigger, conditional fields, and a separate repeatable table of patient feedback rows.

Setting the patient feedback radio

Field: patientFeedbackAvailable (radio — conditional trigger)

Set whether patient feedback is available. This controls which patient-related fields are visible.

// Patient feedback IS available
XfaPdf::feedback()->updateField('patientFeedbackAvailable', 'yes');

// Patient feedback is NOT available
XfaPdf::feedback()->updateField('patientFeedbackAvailable', 'no');

Conditional fields (when "no")

When patient feedback is not available, fill in the date and comment fields:

XfaPdf::feedback()->updateField('patientFeedbackAvailable', 'no');

XfaPdf::feedback()->updateField(
    'lastPatientFeedbackDate',
    'Last patient feedback was collected in November 2022 using '
    . 'the national GMC patient questionnaire.'
);

XfaPdf::feedback()->updateField(
    'patientFeedbackComment',
    'Patient feedback collection was scheduled for October 2024 but '
    . 'postponed due to a change in survey provider. The new GMC patient '
    . 'questionnaire will be distributed in January 2025.'
);

Patient feedback rows (repeatable)

Each row represents one patient feedback exercise or survey. The table appears regardless of the radio selection. Fields per row match the colleague table: role, activity, outcome, supportLocation, firstEntry, amendData.

Text Fields

These standalone text fields sit below both feedback tables.

Commentary on feedback and actions taken

Field: actionsTaken

This field is visible when either colleague or patient feedback is available ("yes"). The doctor describes what they learned from the feedback and what actions they have taken or plan to take as a result.

XfaPdf::feedback()->updateField(
    'actionsTaken',
    'I reviewed colleague and patient feedback with my appraiser. Key themes: '
    . "\n\n"
    . 'Colleague feedback: Overall very positive (4.6/5). Two colleagues '
    . 'suggested I could improve the timeliness of referral letters. I have '
    . 'since implemented a same-day dictation policy and referral letters '
    . 'are now sent within 48 hours of consultation.'
    . "\n\n"
    . 'Patient feedback: Very high satisfaction (4.8/5). Two patients requested '
    . 'longer appointment times for complex problems. I have introduced '
    . 'double-length appointments for annual chronic disease reviews and new '
    . 'patient registrations.'
    . "\n\n"
    . 'Overall, I am pleased with the feedback and will continue to seek '
    . 'opportunities to improve communication and access.'
);

Appraiser's comments

Field: appraiserComments

Written by the appraiser, summarising their view of the feedback and the doctor's response to it. This field mirrors to Section 19 (Appraisal Summary).

XfaPdf::feedback()->updateField(
    'appraiserComments',
    'Dr Smith has collected comprehensive colleague and patient feedback '
    . 'this year with excellent response rates. The feedback is overwhelmingly '
    . 'positive across all domains. She has reflected thoughtfully on the '
    . 'constructive suggestions and has already implemented practical changes '
    . '(same-day dictation policy, double-length appointments). This demonstrates '
    . 'a genuine commitment to continuous improvement in patient care.'
);

Complete Controller Example

Here's a full example filling the entire Feedback section in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    public function fillFeedback()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        $manager = app('xfa-pdf');
        $xfaPdf = $manager->load($pdfPath);

        // -----------------------------------------------
        // 1. Colleague feedback: available
        // -----------------------------------------------
        XfaPdf::setFile($pdfPath);
        XfaPdf::feedback()->updateField('colleagueFeedbackAvailable', 'yes');

        // Add colleague feedback rows
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_10_Feedback',
            'colleague',
            'feedbackRow',
            [
                [
                    'role'            => 'Consultant',
                    'activity'        => 'March 2024 - GMC colleague questionnaire. '
                                       . '18/20 responses from multidisciplinary team.',
                    'outcome'         => 'Mean score 4.6/5. Strong communication and '
                                       . 'leadership. Suggestion to improve referral '
                                       . 'letter turnaround time.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
                [
                    'role'            => 'GP Partner',
                    'activity'        => 'January 2024 - Practice 360 feedback. '
                                       . '6 partners and 4 nurses participated.',
                    'outcome'         => 'Approachable and supportive colleague. '
                                       . 'Effective clinical meeting chair. Valued '
                                       . 'mentoring of nurse prescribers.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
                [
                    'role'            => 'Registrar',
                    'activity'        => 'February 2024 - Feedback from GP registrar '
                                       . 'during ST3 training placement.',
                    'outcome'         => 'Training described as "outstanding". Regular '
                                       . 'tutorials and timely feedback on case write-ups.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
            ]
        );

        // -----------------------------------------------
        // 2. Patient feedback: available
        // -----------------------------------------------
        XfaPdf::feedback()->updateField('patientFeedbackAvailable', 'yes');

        // Add patient feedback rows
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_10_Feedback',
            'patient',
            'feedbackRow',
            [
                [
                    'role'            => 'Patient',
                    'activity'        => 'April 2024 - GMC patient questionnaire. '
                                       . '42/50 responses from consecutive patients.',
                    'outcome'         => 'Mean 4.8/5 for listening and explanation. '
                                       . 'Mean 4.7/5 for shared decision-making. '
                                       . 'Request for longer complex appointments.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
                [
                    'role'            => 'Patient',
                    'activity'        => 'Ongoing 2023-2024 - Friends and Family Test. '
                                       . '156 responses over 12 months.',
                    'outcome'         => '97% recommendation rate. Praised thoroughness '
                                       . 'and empathy. Waiting time comments related '
                                       . 'to practice-level issue.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
                [
                    'role'            => 'Patient',
                    'activity'        => 'June 2024 - Targeted diabetes clinic survey. '
                                       . '25 responses from last 6 months.',
                    'outcome'         => 'Valued structured annual reviews and clear '
                                       . 'HbA1c explanations. Written care plans '
                                       . 'appreciated by several patients.',
                    'supportLocation' => 'Yes',
                    'firstEntry'      => '',
                    'amendData'       => '',
                ],
            ]
        );

        // -----------------------------------------------
        // 3. Text fields
        // -----------------------------------------------
        XfaPdf::feedback()->update([
            'actionsTaken' =>
                'Reviewed colleague and patient feedback with appraiser. '
                . 'Key actions taken:'
                . "\n\n"
                . '1. Implemented same-day dictation policy for referral '
                . 'letters (now sent within 48 hours)'
                . "\n"
                . '2. Introduced double-length appointments for annual '
                . 'chronic disease reviews and new registrations'
                . "\n"
                . '3. Started monthly peer review of consultation skills '
                . 'with practice partner'
                . "\n\n"
                . 'Overall very pleased with feedback. Will continue '
                . 'seeking opportunities to improve.',

            'appraiserComments' =>
                'Dr Smith has collected comprehensive feedback with '
                . 'excellent response rates. Feedback is overwhelmingly '
                . 'positive. She has reflected thoughtfully on constructive '
                . 'suggestions and implemented practical changes. This '
                . 'demonstrates genuine commitment to continuous '
                . 'improvement in patient care.',
        ]);

        // -----------------------------------------------
        // 4. Save the PDF
        // -----------------------------------------------
        $manager->saveXfaPdf($xfaPdf);
        // Or: XfaPdf::save();

        return redirect()->back()->with('success', 'Feedback section saved.');
    }
}
