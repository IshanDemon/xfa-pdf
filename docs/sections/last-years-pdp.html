Section 6: Last Year's PDP

This section reviews the doctor's progress against their previous year's Personal Development Plan. It includes a repeatable table of PDP activities with progress notes, plus general comments and appraiser comments.

Section name: Section_6_LastYearsPDP
Fluent accessor: XfaPdf::lastYearsPdp()
Navigation label: Personal development plans and their review

What's in this section?

FieldTypeWhat it is
roleSelectRole or area of practice the PDP activity relates to (repeatable)
detailMultiline textDescription of the PDP activity or goal (repeatable)
progressMultiline textProgress notes and reflections on the activity (repeatable)
attachmentIDTextInternal attachment reference
fileNameTextAttachment filename
generalCommentsMultiline textGeneral comments about last year's progress
appraiserCommentsMultiline textAppraiser's comments (mirrors to Section 19)

The first three fields (role, detail, progress) are repeatable — they live inside activRpt rows within the reflections container. Each row represents one PDP activity from the previous year. The remaining fields are standalone.

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Reading existing data

Read all fields in this section at once:

// Read everything in Last Year's PDP
$data = XfaPdf::read('Section_6_LastYearsPDP');

// Or using the fluent accessor
$data = XfaPdf::lastYearsPdp()->read();

// $data looks like:
// [
//     'reflections' => [
//         'activRpt' => [
//             ['role' => 'Clinical', 'detail' => 'Complete ALS certification', 'progress' => 'Completed in March...'],
//             ['role' => 'Educational', 'detail' => 'Attend diabetes course', 'progress' => 'Attended 2-day course...'],
//         ],
//     ],
//     'generalComments' => 'Good progress made against all PDP objectives...',
//     'appraiserComments' => '',
// ]

Read a single field:

$comments = XfaPdf::lastYearsPdp()->readField('generalComments');
// 'Good progress made against all PDP objectives...'

PDP Activities (Repeatable Table)

Each row in the PDP table represents one activity or goal from the previous year's development plan. Rows are stored as activRpt elements inside the reflections container. Each row has three fields: role (select), detail (textarea), and progress (textarea).

Adding a new PDP activity

To add a new row with all three fields filled in:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Add a PDP activity row
$xfaPdf = $manager->addRepeatableItem(
    $xfaPdf,
    'Section_6_LastYearsPDP',        // section name
    'reflections',                    // container name
    [
        'role'     => 'Clinical',
        'detail'   => 'Complete Advanced Life Support (ALS) certification to improve emergency response skills in primary care.',
        'progress' => 'Completed ALS provider course in March at Manchester Royal Infirmary. Passed assessment first attempt. Now ALS certified until 2028.',
    ]
);

$manager->saveXfaPdf($xfaPdf);

Adding multiple activities at once

If you need to add several PDP activities, call addRepeatableItem() for each one:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Activity 1: Clinical skills
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_6_LastYearsPDP', 'reflections', [
    'role'     => 'Clinical',
    'detail'   => 'Complete Advanced Life Support (ALS) certification.',
    'progress' => 'Completed ALS provider course in March. Passed assessment first attempt. Now certified until 2028.',
]);

// Activity 2: Educational development
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_6_LastYearsPDP', 'reflections', [
    'role'     => 'Educational',
    'detail'   => 'Attend the RCGP diabetes management update course to improve management of Type 2 diabetes patients.',
    'progress' => 'Attended 2-day residential course in June. Implemented new insulin titration protocol at the practice. HbA1c outcomes improved for 12 patients.',
]);

// Activity 3: Leadership
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_6_LastYearsPDP', 'reflections', [
    'role'     => 'Leadership',
    'detail'   => 'Complete NHS Leadership Academy Edward Jenner programme.',
    'progress' => 'Completed all 8 modules by September. Applied learning to restructure the practice clinical meetings schedule.',
]);

// Activity 4: Research and audit
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_6_LastYearsPDP', 'reflections', [
    'role'     => 'Clinical',
    'detail'   => 'Conduct audit of antibiotic prescribing rates in the practice.',
    'progress' => 'Audit completed in November. Presented findings at practice meeting. Prescribing rates reduced by 15% against baseline.',
]);

$manager->saveXfaPdf($xfaPdf);

Replacing all activities

If you want to clear existing activities and set new ones from scratch, use setRepeatableItems():

$xfaPdf = $manager->setRepeatableItems(
    $xfaPdf,
    'Section_6_LastYearsPDP',        // section
    'reflections',                    // container
    'activRpt',                       // element name
    [
        [
            'role'     => 'Clinical',
            'detail'   => 'Achieve diploma in musculoskeletal medicine.',
            'progress' => 'Enrolled on the 12-month programme. Completed 6 of 8 modules. Expected completion by April next year.',
        ],
        [
            'role'     => 'Educational',
            'detail'   => 'Become a GP trainer by completing the Trainer Approval process.',
            'progress' => 'Submitted portfolio in July. Training practice visit completed in October. Approval confirmed in December.',
        ],
        [
            'role'     => 'Clinical',
            'detail'   => 'Improve palliative care skills through Gold Standards Framework training.',
            'progress' => 'Attended all 6 GSF sessions. Practice now has GSF accreditation. Established monthly palliative care MDT meetings.',
        ],
    ]
);

$manager->saveXfaPdf($xfaPdf);

Reading existing activities

$rows = $manager->getRepeatableItems(
    $xfaPdf,
    'Section_6_LastYearsPDP',
    'reflections'
);

foreach ($rows as $index => $row) {
    echo "Activity {$index}: [{$row['role']}] {$row['detail']}\n";
    echo "  Progress: {$row['progress']}\n\n";
}
// Activity 0: [Clinical] Complete Advanced Life Support (ALS) certification.
//   Progress: Completed ALS provider course in March...
//
// Activity 1: [Educational] Attend the RCGP diabetes management update course...
//   Progress: Attended 2-day residential course in June...

Removing an activity

Remove an activity by its index (0-based):

// Remove the second activity (index 1)
$xfaPdf = $manager->removeRepeatableItem(
    $xfaPdf,
    'Section_6_LastYearsPDP',
    'reflections',
    1
);

$manager->saveXfaPdf($xfaPdf);

Text Fields

Below the PDP activities table, there are two free-text fields. These are standalone (not repeatable).

General comments

Field: generalComments

This is where the doctor reflects on their overall progress against last year's PDP. The form label reads: "Please provide a summary of your progress against last year's PDP."

XfaPdf::lastYearsPdp()->updateField(
    'generalComments',
    'Good progress has been made against all PDP objectives set at last year\'s '
    . 'appraisal. The ALS certification and diabetes course were completed on '
    . 'schedule and have already improved my clinical practice. The antibiotic '
    . 'prescribing audit has led to measurable changes at practice level. '
    . 'The musculoskeletal diploma is ongoing but on track for completion. '
    . 'I feel the objectives were appropriately challenging and relevant to '
    . 'my current roles.'
);

Appraiser's comments

Field: appraiserComments

Written by the appraiser, summarising their assessment of the doctor's progress against last year's PDP. This field mirrors to Section 19 (Appraisal Summary).

XfaPdf::lastYearsPdp()->updateField(
    'appraiserComments',
    'Dr Smith has demonstrated excellent engagement with their PDP. All four '
    . 'objectives have been addressed with clear evidence of completion or '
    . 'significant progress. The antibiotic prescribing audit is particularly '
    . 'impressive, showing a measurable 15% reduction in unnecessary prescribing. '
    . 'The ALS certification and diabetes course directly enhance patient safety. '
    . 'I am satisfied that appropriate progress has been made.'
);

Complete Controller Example

Here's a full example filling the entire Last Year's PDP section in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    public function fillLastYearsPdp()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        $manager = app('xfa-pdf');
        $xfaPdf = $manager->load($pdfPath);

        // -----------------------------------------------
        // 1. Set PDP activity rows
        // -----------------------------------------------
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_6_LastYearsPDP',
            'reflections',
            'activRpt',
            [
                [
                    'role'     => 'Clinical',
                    'detail'   => 'Complete Advanced Life Support (ALS) '
                                . 'certification to improve emergency '
                                . 'response skills in primary care.',
                    'progress' => 'Completed ALS provider course in March '
                                . 'at Manchester Royal Infirmary. Passed '
                                . 'assessment first attempt. Now ALS '
                                . 'certified until 2028.',
                ],
                [
                    'role'     => 'Educational',
                    'detail'   => 'Attend the RCGP diabetes management '
                                . 'update course to improve management '
                                . 'of Type 2 diabetes patients.',
                    'progress' => 'Attended 2-day residential course in '
                                . 'June. Implemented new insulin titration '
                                . 'protocol. HbA1c outcomes improved for '
                                . '12 patients in my care.',
                ],
                [
                    'role'     => 'Leadership',
                    'detail'   => 'Complete NHS Leadership Academy Edward '
                                . 'Jenner programme to develop leadership '
                                . 'skills for PCN Clinical Lead role.',
                    'progress' => 'Completed all 8 modules by September. '
                                . 'Applied learning to restructure the '
                                . 'practice clinical meetings schedule. '
                                . 'Positive feedback from partners.',
                ],
                [
                    'role'     => 'Clinical',
                    'detail'   => 'Conduct audit of antibiotic prescribing '
                                . 'rates in the practice against NICE '
                                . 'guidelines.',
                    'progress' => 'Audit completed in November. Presented '
                                . 'findings at practice meeting. Prescribing '
                                . 'rates reduced by 15% against baseline. '
                                . 'Re-audit planned for next year.',
                ],
            ]
        );

        // -----------------------------------------------
        // 2. Fill text fields
        // -----------------------------------------------
        XfaPdf::setFile($pdfPath);

        XfaPdf::lastYearsPdp()->update([
            'generalComments' =>
                'Good progress has been made against all PDP objectives '
                . 'set at last year\'s appraisal. The ALS certification '
                . 'and diabetes course were completed on schedule and '
                . 'have already improved my clinical practice. The '
                . 'antibiotic prescribing audit led to measurable '
                . 'changes at practice level. I feel the objectives were '
                . 'appropriately challenging and relevant to my roles.',

            'appraiserComments' =>
                'Dr Smith has demonstrated excellent engagement with '
                . 'their PDP. All four objectives have been addressed '
                . 'with clear evidence of completion or significant '
                . 'progress. The antibiotic prescribing audit is '
                . 'particularly impressive, showing a measurable 15% '
                . 'reduction. I am satisfied that appropriate progress '
                . 'has been made against the agreed PDP.',
        ]);

        // -----------------------------------------------
        // 3. Save the PDF
        // -----------------------------------------------
        $manager->saveXfaPdf($xfaPdf);
        // Or: XfaPdf::save();

        return redirect()->back()->with('success', 'Last Year\'s PDP saved.');
    }
}
