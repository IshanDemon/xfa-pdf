Section 4: Scope of Work

This section captures the doctor's current roles, where they work, and any upcoming changes. It includes a repeatable table where each row represents one job or role.

Section name: Section_4_ScopeOfWork
Fluent accessor: XfaPdf::scopeOfWork()

What's in this section?

FieldTypeWhat it is
roleTextJob or role title (e.g. "GP Partner")
detailMultiline textDescription of duties and responsibilities
commencedTextYear the role started (e.g. "2018")
organisationMultiline textOrganisation name and contact details
changesToPracticeMultiline textRelationship between various roles
changesToPracticeNextYearMultiline textExpected changes to scope next year
appraiserCommentsMultiline textAppraiser's comments on scope of work

The first four fields (role, detail, commenced, organisation) are repeatable — they live inside rows that can be added or removed. The last three are standalone text fields.

Setup

Before filling any fields, load the PDF in your controller:

use Xfa\Pdf\Facades\XfaPdf;

// Load the PDF file
XfaPdf::setFile(storage_path('app/xfa-pdf/appraisal.pdf'));

Reading existing data

Read all fields in this section at once:

// Read everything in Scope of Work
$data = XfaPdf::read('Section_4_ScopeOfWork');

// Or using the fluent accessor
$data = XfaPdf::scopeOfWork()->read();

// $data looks like:
// [
//     'scopeOfPracticeStatement' => [
//         'workRpt' => [
//             ['role' => 'GP Partner', 'detail' => '...', 'commenced' => '2018', 'organisation' => '...'],
//             ['role' => 'Clinical Lead', 'detail' => '...', 'commenced' => '2020', 'organisation' => '...'],
//         ],
//     ],
//     'changesToPractice' => 'I hold two roles which complement each other...',
//     'changesToPracticeNextYear' => 'Planning to reduce clinical sessions...',
//     'appraiserComments' => '',
// ]

Read a single field:

$comments = XfaPdf::scopeOfWork()->readField('changesToPractice');
// 'I hold two roles which complement each other...'

Work Rows (Repeatable Table)

Each row in the scope of work table represents one job or role. You can add multiple rows, remove them, or replace all of them.

Adding a new work row

To add a new row with all four fields filled in:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Add a work row
$xfaPdf = $manager->addRepeatableItem(
    $xfaPdf,
    'Section_4_ScopeOfWork',        // section name
    'scopeOfPracticeStatement',      // container name
    [
        'role'         => 'GP Partner',
        'detail'       => 'Providing primary care services including chronic disease management, minor surgery, and on-call duties.',
        'commenced'    => '2018',
        'organisation' => 'Riverside Medical Centre, 42 High Street, Manchester M1 2AB. Tel: 0161 123 4567',
    ]
);

$manager->saveXfaPdf($xfaPdf);

Adding multiple rows at once

If you need to add several roles, call addRepeatableItem() for each one:

$manager = app('xfa-pdf');
$xfaPdf = $manager->load(storage_path('app/xfa-pdf/appraisal.pdf'));

// Row 1: Main clinical role
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', [
    'role'         => 'GP Partner',
    'detail'       => 'Full-time partner. 8 clinical sessions per week. Lead for diabetes and respiratory medicine.',
    'commenced'    => '2018',
    'organisation' => 'Riverside Medical Centre, 42 High Street, Manchester M1 2AB',
]);

// Row 2: Secondary role
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', [
    'role'         => 'Clinical Lead - PCN',
    'detail'       => 'Leading the respiratory pathway across the Primary Care Network. 1 session per week.',
    'commenced'    => '2021',
    'organisation' => 'North Manchester PCN, c/o Riverside Medical Centre',
]);

// Row 3: Teaching role
$xfaPdf = $manager->addRepeatableItem($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', [
    'role'         => 'GP Trainer',
    'detail'       => 'Supervising GP registrars. 1 session per week for training and tutorials.',
    'commenced'    => '2020',
    'organisation' => 'Health Education England, North West Deanery',
]);

$manager->saveXfaPdf($xfaPdf);

Replacing all rows

If you want to clear existing rows and set new ones from scratch, use setRepeatableItems():

$xfaPdf = $manager->setRepeatableItems(
    $xfaPdf,
    'Section_4_ScopeOfWork',        // section
    'scopeOfPracticeStatement',      // container
    'workRpt',                       // element name
    [
        [
            'role'         => 'Salaried GP',
            'detail'       => '6 clinical sessions. Special interest in dermatology.',
            'commenced'    => '2019',
            'organisation' => 'Park View Surgery, Leeds LS1 3AA',
        ],
        [
            'role'         => 'Out of Hours GP',
            'detail'       => 'Weekend and evening shifts. 2 sessions per month.',
            'commenced'    => '2022',
            'organisation' => 'Local Care Direct, Leeds',
        ],
    ]
);

$manager->saveXfaPdf($xfaPdf);

Reading existing rows

$rows = $manager->getRepeatableItems(
    $xfaPdf,
    'Section_4_ScopeOfWork',
    'scopeOfPracticeStatement'
);

foreach ($rows as $index => $row) {
    echo "Row {$index}: {$row['role']} at {$row['organisation']}\n";
}
// Row 0: GP Partner at Riverside Medical Centre...
// Row 1: Clinical Lead - PCN at North Manchester PCN...

Removing a row

Remove a row by its index (0-based):

// Remove the second row (index 1)
$xfaPdf = $manager->removeRepeatableItem(
    $xfaPdf,
    'Section_4_ScopeOfWork',
    'scopeOfPracticeStatement',
    1
);

$manager->saveXfaPdf($xfaPdf);

Text Fields

Below the work table, there are three free-text fields. These are standalone (not repeatable).

Relationship between roles

Field: changesToPractice

This is where the doctor describes how their various roles relate to each other. The form label reads: "Please describe here anything significant regarding the relationship between your various roles."

XfaPdf::scopeOfWork()->updateField(
    'changesToPractice',
    'My GP Partner role and PCN Clinical Lead role are closely related. '
    . 'The PCN work allows me to develop respiratory pathways that directly '
    . 'benefit patients at my practice. My GP Trainer role keeps me up to date '
    . 'with current training standards and evidence-based practice.'
);

Changes to scope next year

Field: changesToPracticeNextYear

Describe any planned changes to the scope of work in the coming year. The form label reads: "Please describe any changes to your scope of work that you envisage taking place in the next year."

XfaPdf::scopeOfWork()->updateField(
    'changesToPracticeNextYear',
    'I plan to reduce my clinical sessions from 8 to 7 per week to allow '
    . 'more time for the PCN Clinical Lead role. I am also considering taking '
    . 'on a CCG advisory role for respiratory services starting April 2025.'
);

Appraiser's comments

Field: appraiserComments

Written by the appraiser, summarising their view of the doctor's scope of work.

XfaPdf::scopeOfWork()->updateField(
    'appraiserComments',
    'Dr Smith has a well-balanced portfolio of clinical and leadership roles. '
    . 'The planned reduction in clinical sessions to accommodate the expanding '
    . 'PCN role is a sensible adjustment. The training role demonstrates a '
    . 'commitment to professional development and supporting the next generation.'
);

Complete Controller Example

Here's a full example filling the entire Scope of Work section in a Laravel controller:

use Xfa\Pdf\Facades\XfaPdf;

class AppraisalController extends Controller
{
    public function fillScopeOfWork()
    {
        $pdfPath = storage_path('app/xfa-pdf/appraisal.pdf');
        $manager = app('xfa-pdf');
        $xfaPdf = $manager->load($pdfPath);

        // -----------------------------------------------
        // 1. Add work rows (roles)
        // -----------------------------------------------
        $xfaPdf = $manager->setRepeatableItems(
            $xfaPdf,
            'Section_4_ScopeOfWork',
            'scopeOfPracticeStatement',
            'workRpt',
            [
                [
                    'role'         => 'GP Partner',
                    'detail'       => 'Full-time partner providing primary care. '
                                    . '8 clinical sessions per week. Lead for '
                                    . 'diabetes and respiratory medicine.',
                    'commenced'    => '2018',
                    'organisation' => 'Riverside Medical Centre, '
                                    . '42 High Street, Manchester M1 2AB. '
                                    . 'Tel: 0161 123 4567',
                ],
                [
                    'role'         => 'Clinical Lead - PCN',
                    'detail'       => 'Leading the respiratory pathway across '
                                    . 'the Primary Care Network. 1 session/week.',
                    'commenced'    => '2021',
                    'organisation' => 'North Manchester PCN, '
                                    . 'c/o Riverside Medical Centre',
                ],
                [
                    'role'         => 'GP Trainer',
                    'detail'       => 'Supervising GP registrars. 1 session/week '
                                    . 'for training and tutorials.',
                    'commenced'    => '2020',
                    'organisation' => 'Health Education England, '
                                    . 'North West Deanery',
                ],
            ]
        );

        // -----------------------------------------------
        // 2. Fill text fields
        // -----------------------------------------------
        XfaPdf::setFile($pdfPath);

        XfaPdf::scopeOfWork()->update([
            'changesToPractice' =>
                'My GP Partner role and PCN Clinical Lead role are closely '
                . 'related. The PCN work allows me to develop respiratory '
                . 'pathways that directly benefit my practice patients. '
                . 'My GP Trainer role keeps me up to date with current '
                . 'evidence-based practice.',

            'changesToPracticeNextYear' =>
                'Planning to reduce clinical sessions from 8 to 7 per week '
                . 'to accommodate expanding PCN responsibilities. Considering '
                . 'a CCG advisory role for respiratory services from April 2025.',

            'appraiserComments' =>
                'Dr Smith has a well-balanced portfolio. The planned session '
                . 'reduction for PCN work is sensible. The training role shows '
                . 'commitment to supporting the next generation of GPs.',
        ]);

        // -----------------------------------------------
        // 3. Save the PDF
        // -----------------------------------------------
        $manager->saveXfaPdf($xfaPdf);
        // Or: XfaPdf::save();

        return redirect()->back()->with('success', 'Scope of Work saved.');
    }
}

Read all data XfaPdf::scopeOfWork()->read()
Read one field XfaPdf::scopeOfWork()->readField('changesToPractice')
Update one field XfaPdf::scopeOfWork()->updateField('changesToPractice', 'text...')
Update multiple fields XfaPdf::scopeOfWork()->update(['field' => 'value', ...])
Get work rows $manager->getRepeatableItems($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement')
Add a work row $manager->addRepeatableItem($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', [...])
Remove a work row $manager->removeRepeatableItem($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', $index)
Replace all rows $manager->setRepeatableItems($xfaPdf, 'Section_4_ScopeOfWork', 'scopeOfPracticeStatement', 'workRpt', [...])
Save changes XfaPdf::save() or $manager->saveXfaPdf($xfaPdf)
