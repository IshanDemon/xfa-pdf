Services

The package uses a microservice architecture. Each service handles one aspect of XFA PDF processing.

PdfBinaryService

Raw PDF binary parsing, stream discovery, decompression, and incremental writes.

load(string $filePath): self

Load a PDF file from disk.

loadBinary(string $binary): self

Load a PDF from a raw binary string.

discoverStreams(): void

Discover all XFA dataset streams. Finds stream...endstream blocks, decompresses via Flate, identifies those containing <xfa:datasets>. Uses declared /Length for extraction. Deduplicates by object number, keeping last occurrence.

decompressStream(string $raw): ?string

Decompress a raw PDF stream. Tries gzuncompress() first, then zlib_decode() with byte stripping.

extractTrailerInfo(): array

Extract trailer info for incremental updates. Returns {prev_xref_offset, root, info, size, id}.

groupConsecutiveKeys(array $map): array

Group consecutive object numbers for xref table generation.

writeIncrementalUpdate(DOMDocument $dom, ?string $outputPath = null): bool

Write modified datasets XML back via incremental update. Appends new objects + xref at end of file.

getDatasetStreams(): array  |  getBinary(): string  |  getFilePath(): string

DatasetService

Reads and writes XFA datasets XML. Uses NamespaceService for namespace-agnostic XPath queries.

extract(PdfBinaryService $pdf): DOMDocument

Extract and parse datasets XML into a DOMDocument.

extractRaw(PdfBinaryService $pdf): string

Extract raw XML string without parsing.

getSections(DOMDocument $dom): array

Get all top-level section names.

getFields(DOMDocument $dom, string $sectionName): array

Get all fields in a section as a nested array.

getFieldValue(DOMDocument $dom, string $fieldPath): ?string

Get a field value by slash-separated path: "Section_3/appraisalYear".

setFieldValue(DOMDocument $dom, string $fieldPath, string $value): DOMDocument

setFieldValues(DOMDocument $dom, array $fields): DOMDocument

Set multiple values. $fields is ['path' => 'value', ...].

toArray(DOMDocument $dom): array

static nodeToArray(DOMNode $node): mixed

Recursively convert a DOM node to array/string.

TemplateService

Parses XFA template XML for field metadata, repeatable definitions, and navigation. The template is never modified.

extract(PdfBinaryService $pdf): ?string

Extract template XML from the PDF binary.

getFieldMetadata(string $templateXml): array

Extract field metadata. Returns array<string, {type, options, caption}>.

getRepeatableSubforms(string $templateXml): array

Identify repeatable subforms (<occur max="-1">). Returns array<string, {fields, min, max}>.

getFieldType(string $templateXml, string $fieldName): ?string

Get UI type of a field (e.g. textEdit, checkButton, choiceList).

getNavigationSections(string $templateXml): array

Extract navigation labels from MenuBar button tooltips. Returns array<int, string>.

RepeatableService

Manages repeatable subform items. Mirrors XFA's instanceManager.addInstance() / removeInstance().

getItems(DOMDocument $dom, string $sectionName, string $container): array

Get all items from a repeatable container. Returns array<int, array<string, string>>.

addItem(DOMDocument $dom, string $section, string $container, array $data): DOMDocument

Add a new item. $data is ['fieldName' => 'value', ...].

removeItem(DOMDocument $dom, string $section, string $container, int $index): DOMDocument

Remove item at index.

updateItem(DOMDocument $dom, string $section, string $container, int $index, array $data): DOMDocument

setItems(DOMDocument $dom, string $section, string $container, string $element, array $items): DOMDocument

Replace all items in a container.

getRowTemplate(string $templateXml, string $container): array

Get field names/types for a new empty row from template.

PreviewService

Generates interactive HTML previews, rendering each XFA field type as its HTML equivalent.

generate(array $sections, array $fieldMeta, array $repeatables, array $sectionLabels): string

Generate a full HTML preview page.

renderSection(string $key, string $label, string $fieldsHtml, bool $isEmpty = false): string

renderFieldsHtml($fields, string $parentPath, array $fieldMeta, array $repeatables): string

static humanize(string $key): string

Convert camelCase / snake_case to readable label.

static resolveFieldMeta(string $dataPath, string $fieldName, array $fieldMeta): array

Resolve metadata with fallback chain. Returns {type, options, caption}.

NamespaceService

Auto-detects XML namespaces, enabling the package to work with any XFA form regardless of vendor.

detect(string $xml): self

Auto-detect all namespaces from an XML string.

registerOnXPath(DOMXPath $xpath): void

Register all detected namespaces on a DOMXPath instance.

getDataNamespace(): ?string

Get the data root namespace (varies per form vendor).

getDataPrefix(): string

getXfaNamespace(): string

getTemplateNamespace(): string

getAll(): array

Get all detected namespaces as prefix => URI.
