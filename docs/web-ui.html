Web UI Guide

Full web interface for uploading, previewing, editing, and downloading XFA PDFs.

Routes

All routes use the configured route_prefix (default: /xfa-pdf) and middleware.

Upload Flow

  1. Navigate to /xfa-pdf/upload
  2. Select an XFA PDF file and optionally provide a document name
  3. The file is stored on the configured disk and a database record is created
  4. Redirected to the preview page

Preview Page

The show route renders a read-only HTML preview. All field types are rendered as HTML equivalents with current values. A sidebar lists all sections with human-readable labels from the template.

Edit Page

The edit route renders an interactive form:

  • All field types rendered with correct HTML inputs
  • Sidebar section navigation with active highlighting
  • Add/remove repeatable items via AJAX
  • Save Changes — writes edits back to the stored PDF
  • Generate & Download — applies edits and returns the PDF

Save vs Download

ActionWhat It Does
Save Changes Saves edits to the stored PDF via incremental update. Stays on page.
Generate & Download Applies current edits and returns the PDF as a download.

Repeatable Items in the UI

Repeatable sections display + and − buttons. These send AJAX requests:

// Add item
fetch('/xfa-pdf/1/add-item', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-CSRF-TOKEN': token
    },
    body: JSON.stringify({
        section: 'Section_7_CPD',
        container: 'cpdList',
        data: { activityType: '', description: '', hours: '' }
    })
});

// Remove item
fetch('/xfa-pdf/1/remove-item', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-CSRF-TOKEN': token
    },
    body: JSON.stringify({
        section: 'Section_7_CPD',
        container: 'cpdList',
        index: 0
    })
});

Customizing Views

Publish views to override the default templates:

php artisan vendor:publish --tag=xfa-pdf-views

Published to resources/views/vendor/xfa-pdf/. Views use the xfa-pdf:: namespace.
